## 🎯 一、 核心设计与项目痛点
在快节奏、高压力的学术与工作环境下，当代年轻人极度缺乏新颖、趣味性的情绪宣泄出口。
本项目打破了传统的枯燥聊天框和死板网页限制，利用 **HTML5 全屏电影画幅 + Web Audio API 声音合成**，打造了一款沉浸感极强的暗黑视觉小说（Visual Novel）怪谈解密游戏。玩家将被困在 4:44 准时发车的午夜地铁中，只有通过搜集线索、洞察隐藏在暗处的致命生存规则，才有可能逃出生天。

---

## 🛠️ 二、 核心关键技能与实现亮点（Core Skills）
本项目在复赛（Prototype）阶段，高标准、100% 跑通了以下核心技术闭环：

1. **影院级电影画幅与响应式动态底图切换（VN Visuals）：**
   彻底移除了传统的手机框限制。游戏画面能够 100% 完美平铺、占满整块浏览器屏幕。底图画面（如昏暗车厢、惨白断手、白光异象、触手古神实体）能够随着玩家的每一次抉择，在 0.6 秒的柔和过渡中无缝进行场景化切换，排版开阔，极富视觉张力。
2. **环境音效实时代码合成引擎（Web Audio Synth）：**
   为了将游戏宽带消耗和加载延迟降到 0，本项目完全通过底层纯代码在浏览器中实时合成“地铁低频震动轰鸣”、“铁轨尖锐摩擦声”以及死亡瞬间的“闪光高频惊悚强音”。无任何外部音频依赖，极大渲染了规则怪谈的情绪价值。
3. **隐藏生存规则判定状态机逻辑：**
   后台静默维护多条致死边界（红色乘客禁忌、乘车票陷阱、车窗窥视深渊等）。玩家的动作会实时触发状态机流转。在不幸触发死亡条件后，屏幕将爆开血红冲击波，并高亮带有呼吸脉冲特效的“循环重置”按钮，防止点击拦截，逻辑无卡顿，允许无限次重试博弈。
